# BlueWave Solutions

Site vitrine statique multi-pages et responsive en francais pour BlueWave Solutions.

## Pages

- `index.html` : accueil et synthese des expertises
- `a-propos.html` : presentation du cabinet, positionnement, publics accompagnes et methodologie
- `domaines.html` : domaines d'intervention et zones geographiques
- `services.html` : offre de services et logique d'intervention
- `contact.html` : prise de contact et formulaire statique

## Structure

- `assets/css/style.css` : styles partages et mise en page responsive
- `assets/js/main.js` : comportement du menu mobile

## Utilisation

Ouvrir `index.html` dans un navigateur ou lancer un serveur local pour visualiser le site.
